## Non-Negotiable Rules for Bolt
1. No two pages share the same primary keyword.
2. Homepage mentions each service once with a link — it does NOT repeat that service's specific keyword density.
3. Related/geo-entity terms (Route 109/110, ZIP 11735, etc.) are for AEO/AI-answer grounding, not ranking density — use naturally, sparingly.
4. Internal linking must use the target page's primary keyword as anchor text where natural — this is what tells Google which page owns which term.
